## 'Buddy' - my first large DIY Project - without tutorials

# About me and my Buddy

- I am a young developer and home-engineer from Germany. In my bedroom, I am building skills and nice little tools since I was 10.  My newest idea, I know I am really not the first with it, was to build a tiny Buddy that accompanied me wherever I go and is good for website hosting and having fun on every trip. At time, I finished with the prototype and I am already working on Version 1.1. You can see all the components from my builds below on the site.

- You can look at the codes and tools in the [Buddy-Releases](Buddy-Releases) folder.

# The Components, I used, for the prototype (V1.0), are:
- ESP32
- 2x Push-Button
- 4x4 Keypad
- Joystick
- 2x OLED Display (128px x 64px)
- Buzzer
- Wires

# Components from V1.1:
- External Camera Module - with LiPo Rechargeable Battery and charger module (USB-C)
- ESP32 C3
- 3x Push-Button
- Slide Switch
- OLED Display (128px x 64px)

# Components from V2.0:
- ESP32 C3
- OLED Display
- 3x Push Button
- Wires
